What these tools do

SPARK TOOLS is a small set of browser-based text tools. Each one gives you a line, prompt, warning, definition, or useful misdirection to test against whatever you are making or thinking through.

The point is not to be right on the first press. The point is to change the angle quickly.

How it works

Pick a tool, use it, then return to the list. Only one tool is visible at a time.

Nothing is saved to a server. Each tool remains a separate self-contained HTML file and can be replaced independently.
