Operator’s note

The Doubt Engine is a small suite of local text machines. Each module does one thing: press a button, get a line, test it against the problem in front of you.

The five finished modules are included. TILT is present as a provisional sixth module and can be replaced later without changing the launcher.

Installed paths

said-it / badvice / idk-machine / bad-evidence / false-positive / tilt.

Each module runs as its own self-contained index.html. Nothing is saved to a server.
