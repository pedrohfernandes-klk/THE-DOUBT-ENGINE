What these tools do

SPARK TOOLS is a shelf of ten single-purpose text machines. One gives you a premise, one a side entrance, one the question under your question, one the case against you, one beautifully terrible advice. Press a button, get a line, test it against whatever you are making, deciding, or stuck inside.

The point is not to be right on the first press. The point is to change the angle fast — and then get back to work.

How it works

Pick a tool, use it, then return to the list. Only one tool is visible at a time.

Nothing is saved to a server. Each tool remains a separate self-contained HTML file and can be replaced independently.
